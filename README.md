# Indoqa Boot BOM

This project contains a Maven Bill of Material (BOM ) for
[Indoqa Boot](https://github.com/Indoqa/indoqa-boot)-based projects.

Among others it sets the versions of following dependencies:
* Spring
* Spark
* Jetty
* Jackson
* slf4j
* log4j
